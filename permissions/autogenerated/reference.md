## Default Permission

Default permissions for offlite plugin - allows all database and sync operations

#### This default permission set includes the following:

- `allow-db-open`
- `allow-db-close`
- `allow-db-execute`
- `allow-db-query`
- `allow-db-batch`
- `allow-db-delete`
- `allow-sync-start`
- `allow-sync-stop`
- `allow-sync-status`
